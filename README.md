# PhonePe-Analysis
This repository contains the analysis of phone-pe data from the year of 2018 to 2024 and the Analysis.
State-wise, district-wise analysis, growth analysis, Top performing states, districts all over india and least performing states,district all over india data provided here.
This structured design allows the application to generate interactive dashboards, maps, KPIs, and analytical visualizations using Streamlit and Plotly.
